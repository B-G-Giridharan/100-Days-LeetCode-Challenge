# 100 Days LeetCode Challenge

A structured, day-by-day approach to solving LeetCode problems over 100 consecutive days, with documented solutions, complexity analysis, and incremental topic coverage.

---

## Challenge Overview

**Goal:** Solve at least one LeetCode problem every day for 100 consecutive days.

| Objective | Description |
|-----------|-------------|
| Consistency | One problem minimum, every single day |
| Understanding | Prefer depth of learning over raw problem count |
| Documentation | Record approach, complexity, and key insights per day |
| Review | Revisit and optimize solutions when possible |

---

## Rules

1. **Daily Commitment** — Solve at least one LeetCode problem every day.
2. **Documentation** — Each solution must include the approach, time complexity, space complexity, and clean code.
3. **No Skipping** — If a day is missed, the challenge restarts.
4. **Quality over Quantity** — Understanding the solution matters more than completing it quickly.
5. **Review** — Revisit previously solved problems to find more optimal approaches.
